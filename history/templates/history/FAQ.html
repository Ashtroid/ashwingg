Introduction

Hi, my name is Ashwin Sethi, and I am a software engineer. I built this website as a tool for people to aesthetically view peoples' League of Legends Ranked 5x5 Solo/Duo Queue match histories without the hassle of ads and messy statistics. Whether it be from your phone, or from your computer, I want you to be able to look at the screen and not have to search or squint for the information you came to see. This project is free, completely out-of-pocket, and open-source.

Our Mission

Ashwin.gg is a League of Legends Match History Visualizer, using the API provided by Riot Games to neatly display all Ranked Solo/Duo Queue games by username. We provide a minimal approach to displaying all the information needed to judge a game without all the clutter. All the code for this project can be found here.

Implemntation

Python calls the Riot API and parses summoner info. The view renders the template with summoner info based on their Ranked Solo/Duo Queue games. AJAX POST calls backend using a View class, returning a rendered String which displays the matches, using standard Python Multiprocessing. Matches are indexed by matchId, which are then stored in the database, and upon compute, check against the database to avoid re-computation for much faster load times and reduced server load. The website is configured to only run through SSL, powered by a LetsEncrypt cert/key pair and enforced by Route 53 and Nginx config. Standard Django headers provide rate limiting to avoid throttling and CSRF protection for security, with in-memory caching to store the rate limiter keys. Static and Media files are served via CDN.
